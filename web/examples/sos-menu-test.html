SOS Menu Test

This tests the SOS.Menu component from the SOS.Ui.js module. Given a particular feature of interest (FOI), say, from a SOS.Map object, it can show all offerings of that FOI. Alternatively, as here, it can show all offerings for all FOIs. Selecting an offering from the Offerings list will then show all observed properties for that offering, in the Observed Properties list. Selecting an observed property can then be used to trigger further behaviour, such as plotting that observed property's data via a SOS.Plot for example. Also, the menu can contain a Controls panel; useful for configuring SOS.Plot or SOS.Table objects.
