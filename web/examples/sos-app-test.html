SOS App Test

This tests the SOS.App default application from the SOS.Ui.js module. The SOS.App pulls together the various UI components, such as SOS.Map, SOS.Plot etc. into an application to enable browsing, visualisation, and access, of SOS data.

Select a station from the map. Select an offering, then an observed property. The observed property's data will then be plotted and tabulated. By default, a rolling month's worth of data is shown, but specific dates can be selected via the Date Range controls under the Controls menu item. To overplot further data, check the Add To Existing control before selecting an additional observed property.
