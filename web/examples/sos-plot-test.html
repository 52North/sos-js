SOS Plot Test

This tests the SOS.Plot component from the SOS.Ui.js module. It requests the air_temperature observed property, for FOI foi_34579 (Halley) from the HalleyMet offering, with a relative time of rollingmonth. It also includes an optional overview plot below the plot, which allows the plot to be subsetted, via a dragged selection across the overview plot. Similarly, a dragged selection can be made on the plot to subselect, which is also reflected in the overview plot. Clicking anywhere on the overview plot resets the subselection. Furthermore, two points can be clicked on the plot, which will then produce summary statistics for the selected interval, displayed in a pop-up dialog.
